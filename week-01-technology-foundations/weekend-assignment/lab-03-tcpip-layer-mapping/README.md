## Objective

Map common networking protocols to their correct TCP/IP model layer.

The TCP/IP model explains how data travels across networks and the internet. It consists of four layers that work together to move data from one device to another.



## TCP/IP Layers

Application Layer  
This layer provides services directly to applications and users.

Transport Layer  
This layer ensures data is delivered between devices.

Internet Layer  
This layer handles addressing and routing of packets.

Network Interface Layer  
This layer handles the physical transmission of data across network hardware.

## Explanation

Each protocol performs a different role within the networking process.

Application layer protocols interact with software applications and provide services such as web browsing and file transfer.

Transport layer protocols manage how data moves between devices and ensure reliability or speed depending on the protocol used.

The Internet layer is responsible for addressing and routing packets between networks.

The Network Interface layer handles how data is physically transmitted across network hardware such as Ethernet or Wi-Fi.
