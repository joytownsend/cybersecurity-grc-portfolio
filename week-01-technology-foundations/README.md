# Week 1 – Technology Foundations

## Overview

Week 1 focused on understanding the basic technology concepts that support cybersecurity and governance risk management.

Topics included:

- computer systems
- networking devices
- storage technologies
- data centers
- cloud computing models

Understanding these systems is important because security professionals must understand how technology works before they can identify risks and vulnerabilities.

## Key Learnings

- Routers direct traffic between networks.
- Switches connect devices within the same network.
- Cloud services allow organizations to scale infrastructure quickly.
- Understanding infrastructure helps identify cybersecurity risks.
